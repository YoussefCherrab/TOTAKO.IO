💎 What is TOTAKO?

TOTAKO is a modern browser-based strategy game inspired by classics like Dots and Boxes.

Built with pure web technologies, it delivers a fast, competitive experience where every move matters.

---

⚡ Why This Game is Different

- 🧠 No Extra Turns — pure strategy
- ⏱️ 5-second decision pressure
- 🤖 Smart AI opponent
- 👥 Local multiplayer (2–4 players)
- 🎯 Instant play — no install

🎮 How To Play

- Click a line between two points
- Complete a diamond to capture it
- No extra turns — plan wisely
- Player with most diamonds wins 🏆

---

🛠️ Tech Stack

- JavaScript
